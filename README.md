# Spotify-Clone
Built with React, Vite and Tailwind I tried to replicate the Spotify Web Player as detailed as possible. I made this one mostly to improve my styling skills.
I took 8 of my favorite songs and added to the tracklist. Every playlist is the same, all have the same 8 songs. You can use the arrow keys to scroll horizontally through both the playlists list and the tracks list while hovering your cursor over it.
You can click on the playlists to open them and on the songs to play them. The play/pause button is functional and toggles when clicked.
Previous and next buttons also work, they follow the order of the playlists. You can also switch to a specific time of the song by clicking on the bar.
You'll need node.js to run this locally, just clone the repository and run it with npm:
git clone https://github.com/julietasegovia/Spotify-Clone
cd Spotify-Clone
npm install (if you don't have it)
npm run dev (it should run at Vite's default port: http://localhost:5173)
